# ProWorkspace Aviso de tratamento de dados

Language: Português

The installer creates database records for organizations, users, user metadata, groups, site mappings, options, and installation state.

Site mappings route domains and optional paths to services. These records may include a domain, path, service name, status, and active identifier.

The first administrator account receives administrative authority for the workspace. Keep this account secure and limit access to trusted operators.

Before production use, configure HTTPS, authentication, backups, monitoring, audit logging, and data retention procedures. Este é um aviso básico para instalação auto-hospedada. Substitua antes do uso em produção.
