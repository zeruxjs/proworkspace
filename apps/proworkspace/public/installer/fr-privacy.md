# ProWorkspace Avis de confidentialité

Language: Français

ProWorkspace stores organization and administrator information needed to create and operate the first workspace.

During installation, records may include organization name, email domain, email policy, selected email users, administrator name, administrator email, service site mappings, active identifiers, and installation state.

The administrator password is hashed before storage. The plain password is not intentionally stored by ProWorkspace.

Only enter information you are authorized to manage. Ceci est un avis de base pour une installation auto-hébergée. Remplacez-le avant la production.
